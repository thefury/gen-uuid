# gen-uuid

gen-uuid is a small utility program to generate UUIDs on the command line.

## Installation

```bash
go install github.com/thefury/uuid-gen
```

## Usage

```bash
> gen-uuid

432e88ec-8fbf-426c-942f-0cfbab39d5c1
```
